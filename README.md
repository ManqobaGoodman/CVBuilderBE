# CVBuilderBE

#Backend (Spring Boot) – README.md
Project Name

AI CV Builder API

Overview

This is a Spring Boot backend application that integrates with Amazon Web Services AWS Bedrock to generate structured CV data from user input.

The API accepts raw text or structured input and returns a clean, editable CV format.

#Features
AI-powered CV generation
Integration with AWS Bedrock (Claude model)
REST API for frontend consumption
JSON-based structured CV output
Error handling & validation
#Tech Stack
Java 17+
Spring Boot
AWS SDK (Bedrock Runtime)
Maven
